# Homeland - De Pátria para Pátria

## Descrição

O projeto Homeland é uma galeria de arte virtual responsiva que celebra as origens e jornadas dos profissionais de tecnologia da TripleTen. Ele exibe histórias, citações e fotografias de paisagens de diversas partes do mundo, mostrando que não importa a sua origem, todos são bem-vindos.

## Tecnologias e Técnicas Utilizadas

- HTML5 (Semântico)
- CSS3
- Metodologia BEM (Block Element Modifier) com estrutura de arquivos Flat
- Flexbox e CSS Grid Layout para alinhamento, distribuição e galeria responsiva
- Media Queries para Design Responsivo (Desktop 1280px, Tablet 768px, Mobile 320px)
